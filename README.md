# 🐦 AI Tweet Generator

An AI-powered tweet generation tool built using Python and Groq Llama 3 that creates short, engaging, and motivational tweets based on user-provided topics.

---

## 🚀 Features

* Generate motivational tweets instantly
* Uses Groq Llama 3.3 70B for content generation
* Keeps tweets concise and engaging
* Simple command-line interface
* Custom topic input

---

## 🛠️ Tech Stack

* Python
* Groq API
* Llama 3.3 70B

---

## 📂 Workflow

User Topic
↓
Prompt Creation
↓
Groq Llama 3
↓
Tweet Generation
↓
Display Output

---

## 💡 Example

Input:

```text
discipline
```

Output:

```text
Success isn't built on motivation alone. It's built on discipline—the ability to keep moving forward even when you don't feel like it. Small consistent actions create extraordinary results. 🚀
```
